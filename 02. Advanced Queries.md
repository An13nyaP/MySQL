## 1ï¸âƒ£ Subqueries (Nested SELECTs)

A subquery is a query inside another SQL query. It helps you perform operations in multiple steps logically.

```sql
-- Find employees who earn more than the average salary
SELECT name, salary
FROM employees
WHERE salary > (
    SELECT AVG(salary) FROM employees
);
```

ğŸ“Œ *Use Case:* Filter based on dynamic conditions like averages, counts, or data from related tables.
